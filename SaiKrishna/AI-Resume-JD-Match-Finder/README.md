This project is a Python-based desktop application that matches a candidate's resume
with a job description using NLP techniques.

## Features
- Resume upload (PDF / DOCX)
- Skill-based matching
- Experience-based evaluation
- Final match percentage with explanation
- Offline, no API keys required
- Tkinter-based GUI

## Technologies Used
- Python
- Tkinter
- Scikit-learn
- NLP (TF-IDF, rule-based extraction)

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python main.py

## Use Case
Automates resume screening by comparing skills and experience with job requirements.
